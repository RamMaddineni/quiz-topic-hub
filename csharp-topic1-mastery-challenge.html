Copying and parameter passing

Questions 1–4 · 4 points

1. What is printed when this struct is passed to a normal parameter? 
struct Counter { public int Value; }

static void Increment(Counter counter)
{
    counter.Value++;
}

var counter = new Counter { Value = 10 };
Increment(counter);
Console.WriteLine(counter.Value);

2. What is printed after this reference-type argument is mutated and then reassigned? 
class User { public string Name = ""; }

static void Update(User user)
{
    user.Name = "B";
    user = new User { Name = "C" };
}

var user = new User { Name = "A" };
Update(user);
Console.WriteLine(user.Name);

3. What changes when the same reference variable is passed with ref? 
class User { public string Name = ""; }

static void Replace(ref User user)
{
    user = new User { Name = "C" };
}

var user = new User { Name = "A" };
Replace(ref user);
Console.WriteLine(user.Name);

4. Which statement correctly distinguishes ref, out, and in?
